# react-app

1. node.js, npm のインストール
   - [node.js](https://nodejs.org/ja)
   - `npm install -g npm@latest`
2. プロジェクトの作成
   - `npm create vite@latest`
     ```
     Project name:
     > {アプリ名}
     Select a framework:
     > React
     Select a variants:
     > TypeScript
     ```
3. パッケージのインストール
   - `npm install`
4. 実行
   - `npm run dev`
